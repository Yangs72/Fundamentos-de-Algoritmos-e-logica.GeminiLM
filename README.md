# 🧠 Fundamentos de Algoritmos e Lógica com NotebookLM

> Base de conhecimento construída com NotebookLM para estudar algoritmos, lógica de programação e estruturas de dados utilizando IA Generativa.

---

## 📖 Sobre o projeto

Este projeto documenta a criação de uma base de conhecimento no NotebookLM voltada ao estudo de algoritmos e lógica de programação.

O objetivo é reunir livros, documentações e materiais técnicos em um único ambiente, permitindo realizar pesquisas, obter explicações fundamentadas, comparar algoritmos e revisar conceitos de forma organizada.

Além de servir como ferramenta de estudo, este repositório demonstra minhas habilidades em documentação técnica, organização do conhecimento e uso de IA Generativa aplicada à aprendizagem.

---

## 🎯 Objetivos

- Organizar conteúdos sobre algoritmos e lógica de programação.
- Utilizar NotebookLM como assistente de estudos.
- Documentar a metodologia utilizada.
- Compartilhar exemplos e materiais produzidos durante os estudos.
- Desenvolver um projeto de portfólio para a área de Tecnologia da Informação.
