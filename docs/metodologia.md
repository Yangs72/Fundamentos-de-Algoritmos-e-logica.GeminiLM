# Metodologia

## Objetivo

Construir uma base de conhecimento utilizando o NotebookLM para centralizar conteúdos sobre algoritmos, lógica de programação e estruturas de dados.

---

## Etapas de construção

### 1. Seleção das fontes

Foram selecionados livros, apostilas e documentações técnicas relacionadas aos principais fundamentos da programação.

Os materiais foram escolhidos buscando qualidade, confiabilidade e abrangência dos conteúdos.

---

### 2. Organização

As fontes foram importadas para o NotebookLM e organizadas em uma única base de conhecimento.

Essa organização permite consultar diferentes materiais simultaneamente.

---

### 3. Interação com a IA

O NotebookLM foi utilizado para:

- responder dúvidas;
- resumir capítulos;
- comparar algoritmos;
- explicar conceitos;
- revisar conteúdos;
- criar exercícios.

---

### 4. Documentação

Todo o processo foi documentado neste repositório para demonstrar a metodologia utilizada na construção da base de conhecimento.

---

## Fluxo de trabalho

```text
Selecionar fontes
        ↓
Importar para o NotebookLM
        ↓
Organizar materiais
        ↓
Realizar consultas
        ↓
Produzir resumos
        ↓
Documentar aprendizados
```

---

## Resultado esperado

Uma base de conhecimento organizada que auxilie no aprendizado de algoritmos e lógica de programação, reduzindo o tempo gasto procurando informações em diferentes materiais.
