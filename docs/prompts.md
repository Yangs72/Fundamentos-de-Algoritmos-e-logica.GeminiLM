# Biblioteca de Prompts

Este documento reúne os principais prompts utilizados para explorar a base de conhecimento criada no NotebookLM.

---

# 1. Explicação de conceitos

## Objetivo

Obter explicações claras e objetivas.

### Prompt

Explique este conceito como se eu fosse um estudante iniciante de programação.

---

# 2. Comparação

## Objetivo

Comparar algoritmos.

### Prompt

Compare os algoritmos apresentados nas fontes destacando:

- vantagens
- desvantagens
- complexidade
- quando utilizar cada um

---

# 3. Revisão

## Objetivo

Revisar rapidamente um conteúdo.

### Prompt

Resuma este capítulo em tópicos mantendo apenas as informações essenciais.

---

# 4. Exercícios

## Objetivo

Praticar os conceitos estudados.

### Prompt

Crie cinco exercícios sobre este conteúdo.

Não forneça as respostas imediatamente.

---

# 5. Correção

## Objetivo

Validar o entendimento.

### Prompt

Faça perguntas sobre este assunto e corrija minhas respostas explicando meus erros.

---

# 6. Aplicação prática

## Objetivo

Relacionar teoria e prática.

### Prompt

Mostre exemplos reais onde este algoritmo é utilizado na programação.

---

# Considerações

Os prompts foram desenvolvidos para tornar o NotebookLM um ambiente de aprendizagem ativo, favorecendo compreensão, revisão e aplicação prática dos conceitos estudados.
